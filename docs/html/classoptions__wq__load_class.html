Ausführliche Beschreibung

Konfigurationsklasse für WQ-Load Berechnungen.

Erbt von optionsClass und fügt Parameter für Belastungsberechnungen hinzu, einschließlich Pfade zu Eingabedaten, Szenario-IDs und Timing-Parameter.
