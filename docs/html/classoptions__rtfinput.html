Ausführliche Beschreibung

Konfigurationsklasse für Return Flow Input.

Diese Klasse verwaltet Pfade und Parameter für die Verarbeitung von verschiedenen Rückfluss-Quellen:

  • Industrielle Abwässer (manufacturing, cooling water)
  • Bewässerungs-Rückfluss
  • Korrekturfaktoren für Rückfluss-Berechnungen

Definiert in Zeile 24 der Datei options_rtf_input.h.
