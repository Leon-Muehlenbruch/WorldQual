Ausführliche Beschreibung

Konfigurationsklasse für das Kopieren von Szenarien.

Diese Klasse verwaltet Parameter zum Kopieren von Referenz-Szenarien zu neuen Szenario-IDs, mit Support für Bereichs-basiertes Kopieren.
