Ausführliche Beschreibung

Speichert statistische Daten für einzelne Messstationen.

Diese Klasse enthält alle relevanten Informationen für die statistische Auswertung an Messstationen, einschließlich:

  • Berechnete Konzentrationen
  • Gemessene Werte
  • Unsicherheits-Flags
  • Leitfähigkeitsmessungen

Im Gegensatz zu statisticClass fokussiert diese Klasse auf punktuelle Stationsmessungen anstatt auf Flussabschnitte.
