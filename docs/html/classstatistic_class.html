Ausführliche Beschreibung

Speichert statistische Daten für einen Flussabschnitt.

Diese Klasse enthält alle relevanten Informationen für die statistische Auswertung und Validierung von Modellergebnissen, einschließlich:

  • Berechnete Konzentrationen und Belastungen
  • Gemessene Werte von Messstationen
  • Abflussdaten (Q)
  • Flussabschnitts-Geometrie
