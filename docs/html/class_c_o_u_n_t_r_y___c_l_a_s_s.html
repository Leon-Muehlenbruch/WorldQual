Ausführliche Beschreibung

Verwaltet Länderinformationen für ein bestimmtes Szenario und Jahr.

Diese Klasse liest Länder-IDs (ISO-Nummern) aus der Datenbank und stellt Methoden zur Verfügung, um zwischen Länder-Positionen im Array und ISO-Nummern zu konvertieren.
