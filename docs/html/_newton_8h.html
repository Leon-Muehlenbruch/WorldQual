Ausführliche Beschreibung

Newton-Verfahren zur Kurvenanpassung (Curve Fitting).

Autor
Ellen Teichert

Implementiert das Newton-Raphson-Verfahren zur Anpassung von Funktionen an gegebene Datenpunkte. Wird hauptsächlich für Wassertemperatur-Kurven verwendet.

Approximationsfunktion: f(x) = c0 / (1 + exp(c1*x + c2))
