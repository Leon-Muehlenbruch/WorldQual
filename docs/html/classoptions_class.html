Ausführliche Beschreibung

Abstrakte Basisklasse für Programmoptionen.

Verwaltet grundlegende Konfigurationsparameter wie Datenbankzugriff, Input-Quellen (DB vs. UNF-Dateien) und Kontinent-Spezifikationen. Wird von spezialisierten Options-Klassen erweitert.
